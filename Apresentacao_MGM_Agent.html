🤖 MGM Agent

Processador Inteligente de Indicações

⚙️ Como Funciona (Fluxo Detalhado)

1
ENTRADA: Usuário Preenche Formulário

Vendedor acessa http://localhost:3000 e cola múltiplos telefones em lote (um por linha), seleciona origem e proprietário responsável.

↓
2
NORMALIZAÇÃO: Sistema Padroniza Dados

MGM Agent normaliza todos os telefones para formato E.164 internacional. Ex: "(11) 98765-4321" → "+5511987654321"

↓
3
BUSCA INTELIGENTE: Detecta Duplicados

Sistema busca no HubSpot: contato com esse telefone já existe? Se SIM → atualiza. Se NÃO → cria novo.

↓
4
SINCRONIZAÇÃO: Contato Criado/Atualizado

Contato é criado ou atualizado no HubSpot com propriedades customizadas (origem, data, proprietário, etc).

↓
5
AUTOMAÇÃO: Deal Criado Automaticamente

Workflows HubSpot disparam automaticamente e criam deal no pipeline MGM. Sem trabalho manual!

↓
6
RESULTADO: Exibição em Tempo Real

Sistema retorna resultado para cada telefone: ✅ Criado/Atualizado ou ❌ Erro. Tudo em segundos!

🔗 Workflows Inteligentes HubSpot

O sistema possui 2 workflows automáticos no HubSpot que trabalham juntos para garantir zero duplicação e máxima sincronização de dados:

A
Workflow 1: Deal Creation

Quando: Novo contato criado com indicação MGM
O que faz: Cria deal automaticamente no pipeline MGM
Resultado: Vendedor vê deal pronto para trabalhar

+
B
Workflow 2: Associate Contacts (O Diferencial! 🌟)

Quando: Contato é criado por FORA (signup, form, API externa, etc) com mesmo telefone de um contato MGM
O que faz: Detecta o duplicado e associa ao contato MGM + deal já existente
Resultado: Histórico unificado, sem deals duplicados, sem perda de dados

💡 Por que é importante: Indica que o prospect pode chegar de várias fontes (landing page, formulário de contato, integração parceira, etc). O sistema MGM não "perde" o contato - ele automaticamente integra no pipeline de vendas e associa ao deal correto!

🧪 Como Testar Agora

Opção 1: Produção (Sem Instalação)

  1. Acesse: https://mgm-agent-production.up.railway.app
  2. Veja a interface em tempo real
  3. Cole um telefone para testar (ex: (11) 98765-4321)
  4. Clique "Processar" e veja resultado em segundos

Opção 2: Local (5 minutos)

  1. Clone: git clone https://github.com/StefanyBertolacini/mgm-agent.git
  2. Entre na pasta: cd mgm-agent
  3. Instale: npm install
  4. Configure .env com sua HubSpot API Key
  5. Rode: npm start
  6. Acesse: http://localhost:3000
